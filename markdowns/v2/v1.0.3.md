# Changelog

## 1.0.3 - 2015-11-28

### What's Changed

### Added

- require PHP 8+
- drop support for PHP 7.x
- use PHP 8 syntax where possible

### Deprecated

- urlencode() new query params values
- Added authorization bearer for each endpoint request
- support PHP 9
